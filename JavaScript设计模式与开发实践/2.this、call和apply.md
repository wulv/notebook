### 2.1 this
跟别的语言大相径庭的是， JavaScript 的 this 总是指向一个对象，而具体指向哪个对象是在运行时基于函数的执行环境动态绑定的，而非函数被声明时的环境。
#### 2.1.1 this的指向
除去不常用的 with 和 eval 的情况，具体到实际应用中， this 的指向大致可以分为以下 4 种。
- 作为对象的方法调用。

当函数作为对象的方法被调用时， this 指向该对象：
```
var obj = {
    a: 1,
    getA: function(){
      alert ( this === obj ); // 输出： true
      alert ( this.a ); // 输出: 1
    }
};
obj.getA();
```
- 作为普通函数调用
